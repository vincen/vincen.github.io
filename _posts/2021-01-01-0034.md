# 数据通信基本原理及 IP 地址的规划和设计

## 企业网络架构介绍

### 大型企业网络结构

```wiki
设备 —> 接入层 —> 汇聚层 —> 核心层 —-> 内部服务器集群
                                +-> Internet
```

接入层：主要保证向下接入的数量。

汇聚层：主要作用对流量进行甄别和分类，作各种策略，例如哪些流量可以通过，哪些流量不能通过。

核心层：把汇聚层上来的流量，提供快速转发的功能。

大型企业网络的设计思想：一定要分层次化，把流量分类，按照层次来进行数据传递，提供设备的冗余。

## 传输介质简介

### 常见传输介质

| 以太网标准  | 类别              | 最长有效传输距离 |
| ----------- | ----------------- | ---------------- |
| 10BASE5     | 粗同轴电缆        | 500米            |
| 10BASE2     | 细同轴电缆        | 185米            |
| 10BASE-T    | 两对3/4/5类双绞线 | 100米            |
| 100BASE-TX  | 两对5类双绞线     | 100米            |
| 1000BASE-T  | 四对5e类双绞线    | 100米            |
| 10BASE-F    | 单模/多模光纤     | 2000米           |
| 100BASE-FX  | 单模/多模光纤     | 2000米           |
| 1000BASE-LX | 单模/多模光纤     | 316米            |
| 1000BASE-SX | 多模光纤          | 316米            |

多模光纤：光源的模间色散比较大，传播距离近。传播越远色散越大，因为损耗越大。一般局域网，高带宽的链接，适合多模光纤。

单模光纤：长距离传播一般使用。

CSMA/CD：解决共享型介质网络中的冲突域。
